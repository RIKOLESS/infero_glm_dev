# playbook · 应急力量预置

> 适用触发词：预置、前置、驻防、力量部署、警力预置、路政预置。
> 场景对齐：比赛场景 5（应急力量预置）。
> 推荐报告模板：`reports/decision_memo.md`（决策建议书）+ `reports/duty_brief.md`（值班简报）。

---

## 一、预报（forecast）

**目标**：在灾情发生前，把救援力量 + 物资 + 关注岗位提前部署到潜在高影响路段。

数据源与调用：

- `await window.weatherPlatform.summary({ province })`：拿预警态势 + 短临排行 + 风险路段。
- `await window.weatherPlatform.resources({ province })`：救援点 + 物资库 + 风险路段 + 隐患点。
- `await window.weatherPlatform.segmentRisk({ province })`：连续降雨风险区（Class1/2）。

关键判定阈值（可根据实际情况调整）：

- 未来 6h 预计有 **红色降雨**：整省进入 II 级预置准备。
- 未来 6h 预计有 **橙色降雨** 叠加 Class1 风险区：区县级 III 级预置。
- 未来 24h 出现 **雨雪冰冻** 或 **大雾** 组合：走 IV 级预置 + 加密巡查。

---

## 二、研判（analyze）

按步骤：

1. **高影响路段清单**：从 `shortTermRank / highImpactSegments / affectedTop10` 归并去重取 TOP 5–10。
2. **风险叠加**：每条路段标注是否位于 `riskZoneClass1/2`、是否命中 `hazardPoints`（桥梁 / 隧道 / 边坡 / 排水）。
3. **力量-路段匹配**：
   - 对每条重点路段，从 `vehicleRescuePoints` 里找最近 1–2 处（直线距离）。
   - 对每条重点路段，从 `materialReserves` 里找匹配灾种的最近 1 处物资库。
4. **岗位建议**：
   - 值班加密：从"3 班倒"改"AB 岗联守"。
   - 巡查加密：路政沿线巡查频次由 4 次/天 → 6 次/天（示例值，可调）。
   - 关键节点驻守：桥梁、隧道口、易积水区。

---

## 三、决策（decide）

输出预置方案骨架：

```
应急力量预置建议 v1
生效时间：<时间窗口>
预置等级：<II/III/IV>

一、重点关注路段（TOP N）
1. <路线 · 桩号 · 类型（如"位于连续降雨一类风险区"）>
2. ...

二、力量部署建议
- 救援点：<救援点名>（距离 ~X km），承担 <路段 A / B> 施救
- 物资库：<库名>（距离 ~Y km），预置 <品类 数量（经验值）>
- 巡查加密：<路段范围> 巡查频次由 x → y
- 关键节点驻守：<桥梁/隧道/易积水区> 建议 2 人驻守

三、通报对象
- 内部：<路政 / 交警 / 应急管理 / 属地政府>
- 上级：<路网中心 / 省高指挥中心>

四、成效评估计划
- 每 3h 复核短临排行，动态调整
- 事件后按 after_action.md 复盘
```

---

## 四、处置（act）

db 侧动作：

1. `renderStatus(bundle)`：Canvas 展示预置候选清单。
2. 出 `reports/decision_memo.md` 决策建议书 + `reports/duty_brief.md` 值班简报。
3. `openBigScreen('threeMap')` 便于空间叠加复核。
4. 与 `vehicle_rescue.md` / `material_dispatch.md` 交叉引用，避免重复推荐同一救援点。

---

## 五、评估（review）

事件结束后：

- **预置命中率**：TOP N 路段是否真的发生了阻断/事故；未发生的路段是否属于误报（可接受一定量误报，避免漏报）。
- **响应时长**：从预警发布 → 力量到位的耗时。
- **物资使用率**：预置物资的实际使用比例，太低说明预置过重，太高说明预置量不足。
- **改进项**：接口延迟、匹配偏差、通报断点。
- 用 `reports/after_action.md` 归档。

---

## 附：常见坑与提醒

- **不要重复推荐**：同一救援点被多个路段占用时，明确标注"承担多路段"，避免值班主任误以为可无限承担。
- **经验值需持续校准**：巡查频次、驻守人数是模板值，实际靠单位历史统计。
- **数据缺失时**：如某接口空数据，明说"该维度平台本轮无数据，力量部署以人工经验为准"。
